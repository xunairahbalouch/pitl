# Pakistan to Ivy League

A premium, interactive website for the Pakistan to Ivy League nonprofit.

## Theme
- **Olive green** — Primary accent
- **Cream white** — Background
- **Black** — Accents and borders

## Pages
1. **Home** — Hero with "Launching Soon," interactive Pakistan map, Ivy League crests
2. **Our Mission** — Educational content about the nonprofit's purpose and offerings
3. **Pathways** — Step-by-step guide to Ivy League applications

## Features
- Interactive Pakistan map with hover tooltips (regions: Kashmir, Punjab, Sindh, Balochistan, KPK)
- Ivy League crest hover effects
- Scroll-triggered reveal animations
- Responsive navigation with mobile menu
- Clean, premium typography (Cormorant Garamond + Outfit)

## Run Locally
Open `index.html` in a browser, or use a local server:

```bash
npx serve .
```
